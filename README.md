# Heart Disease Data Analysis

# Project Overview

This project focuses on analyzing heart disease data to identify key risk factors and patterns using Python, Power BI, and MySQL.

# Objectives

* Analyze patient health data
* Identify key factors affecting heart disease
* Build interactive dashboards
* Generate insights using SQL queries

# Tools & Technologies

* Python (Pandas)
* Power BI
* MySQL
* Excel

# Project Workflow

1. Data Cleaning using Pandas
2. Data Transformation (converted coded values) using Excel
3. Exploratory Data Analysis (EDA)
4. Dashboard creation in Power BI
5. SQL-based analysis

# Key Insights

* Heart disease is more common in older age groups
* High cholesterol and BP increase risk
* Males are more affected than females

# Files Included

* Raw dataset
* Cleaned dataset
* Python notebook
* Power BI dashboard
* SQL queries

## 🚀 Conclusion

This project demonstrates end-to-end data analysis from raw data to actionable insights.
